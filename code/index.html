CS祭とは？

　東京造形大学の芸術祭である「CS祭」は、学生のみで構成されているCS祭実行委員会が企画・運営を行っています。
　今年度は、10月18日/19日/20日の3日間に渡り、開催されます。
　学生たちの有り余るほどの情熱やアイディア満載の作品展示、マーケット、パフォーマンス、模擬店、イベントなどをご用意して皆さまをお待ちしています。 ぜひご来場ください！
